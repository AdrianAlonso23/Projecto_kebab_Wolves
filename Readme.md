# Proyecto Web Restaurante

## Descripción del proyecto
Este proyecto consiste en una **aplicación web de comercio de restauración**. Permite a los usuarios seleccionar productos de un restaurante, agregarlos a un carrito, realizar pedidos y consultar su historial.  
Además, incluye un **panel de administración** donde el administrador puede gestionar productos, usuarios, pedidos y ofertas mediante un **API propia**.

El proyecto está desarrollado usando:  
- **PHP**, **HTML**, **CSS** y **Bootstrap** para la web.  
- **JavaScript** para funcionalidades dinámicas del panel admin.  
- **Modelo-Vista-Controlador (MVC)** para estructurar la aplicación.  
- **Base de datos MySQL** dentro de un contenedor Docker.  
- **API externa** para conversión de monedas (FreeCurrencyAPI).

## Uso de la aplicación

### Como usuario
- Navegar por los productos y agregarlos al carrito.  
- Confirmar pedidos con cálculo automático de total e impuestos.  
- Consultar historial de pedidos y estado de cada uno.  

### Como administrador
- Acceder al panel admin mediante rol de **Administrador**.  
- CRUD completo de productos y usuarios.  
- Crear, modificar y eliminar pedidos.  
- Visualizar pedidos y convertir su total a diferentes monedas.  

---

## Características principales
- Gestión de productos y usuarios mediante **API REST**.  
- Carrito de compra persistente usando **sessionStorage**.  
- Ofertas y descuentos automáticos en productos.  
- Conversión de precios a múltiples monedas.  
- Diseño **responsive** para escritorio y móvil.
